# Mobilidade Inclusiva 🚗♿

Projeto front-end desenvolvido em React com o objetivo de apresentar uma plataforma voltada para **acessibilidade, mobilidade e inclusão**, conectando pessoas a soluções que facilitam o deslocamento no dia a dia.

---

# 📌 Sobre o projeto

O **Mobilidade Inclusiva** é um site que apresenta informações sobre uma proposta de plataforma de mobilidade acessível.

A aplicação possui páginas que explicam o funcionamento do serviço, os planos disponíveis e as medidas de segurança da plataforma.

---

# 🧭 Páginas do site

O projeto possui as seguintes páginas:

* **Home** – página inicial com apresentação da proposta do projeto
* **Sobre** – informações sobre o objetivo da plataforma
* **Planos** – apresentação dos planos disponíveis
* **Segurança** – informações sobre proteção e segurança dos usuários
* **Cadastro** – área destinada ao cadastro de usuários e motoristas

---

# 🛠 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

* React
* JavaScript
* HTML5
* CSS3
* React Router
